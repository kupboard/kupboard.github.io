How to deploy collection?#

Package#

Deployment of a default package

$ kupboard collection package -n <package-name> [-a <action-name>]

Deployment of all default packages

$ kupboard collection package

Deployment of a customer package

$ kupboard collection package -c <collection-name> -n <package-name> [-a <action-name>]

Application#

Build customer application

$ kupboard collection app -c <collection-name> -n <app-name>

Build all customer applications

$ kupboard collection app -c <collection-name>

Build all customer applications from all customer collections

$ kupboard collection app
