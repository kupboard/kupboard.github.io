CLI

build#

build is a command used to build an OpenAPI based application.

Usage:
kupboard build [flags]
Flags:
--app-port string application port (default "8080")
-h, --help help for build
--mock mock
-p, --push push application image (default true)
-r, --replica string replication size (default "3")
-s, --service string service name
--svc-port string service port (default "8080")
-v, --version string application version (default "1.0")

kollection#

kollection is for management of packages and application included in kollections. It has two sub commands, app and package.

Usage:
kupboard kollection [command]
Available Commands:
app Build and deploy kollection applications
package Install packages on remote clusters

app#

app is a sub command of kollection and used to build an application in a kollection.

Usage:
kupboard kollection app [flags]
Flags:
-c, --kollection string target kollection (default "default")
-h, --help help for app
-n, --name string application name ('all' for building all applications)

package#

package is a sub command of kollection and used to excute actions defined in a config.

Usage:
kupboard kollection package [flags]
Flags:
-a, --action string action name (default "default")
-c, --kollection string target kollection (default "default")
-h, --help help for package
-n, --name string package name

deploy#

deploy is for deployment of application containers on target clusters.

Usage:
kupboard deploy [flags]
Flags:
--delete delete applications
-h, --help help for deploy
--mock use mockup
-s, --service string service name

setup#

setup is to initialize clusters.

Usage:
kupboard setup [flags]
Flags:
-h, --help help for setup
--init-cluster initialize targer cluster
--init-finish finish initialization
--init-k8s install kubernetes
--init-user initialize user settings
--keygen create ssh keys
--root-password string root password for init-user
--root-username string root username for init-user

ssh#

ssh is for ssh connection to a host.

Usage:
kupboard ssh <hostname> [flags]
Flags:
-c, --command string command string
-h, --help help for ssh
